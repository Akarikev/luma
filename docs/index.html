Luma

A compact tray-first desktop app for focus sessions, tasks, clipboard history, and a quick glance at system stats—built with Tauri and React.

Features

  • Pomodoro-style focus timer with break sessions
  • Tasks with optional reminders
  • Clipboard history
  • Lightweight system stats
  • Settings: simple task mode, notifications, panels, data folder
